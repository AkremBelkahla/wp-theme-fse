Un thème WordPress FSE minimaliste et performant, conçu pour les agences digitales.

== Description ==

MinimalFSE est un thème WordPress Full Site Editing (FSE) minimaliste, performant et moderne, conçu pour les agences digitales et les sites vitrines professionnels. Il utilise Tailwind CSS pour une mise en page flexible et efficace.

= Caractéristiques principales =

* Full Site Editing complet
* Design minimaliste et élégant
* Haute performance avec CSS optimisé
* Entièrement responsive
* Palette de couleurs personnalisable
* Typographies personnalisables
* Compositions (patterns) prédéfinies
* Variations de blocs personnalisées
* Support des blocs personnalisés
* Compatible avec Advanced Custom Fields

= Compositions incluses =

* Section Héro avec arrière-plan
* Grille de services
* Section de témoignages
* Appel à l'action
* Présentation d'équipe

== Installation ==

1. Dans votre administration WordPress, allez dans Apparence > Thèmes > Ajouter
2. Cliquez sur "Téléverser un thème"
3. Sélectionnez l'archive du thème et cliquez sur "Installer maintenant"
4. Une fois l'installation terminée, cliquez sur "Activer"

== Personnalisation ==

Ce thème utilise le Full Site Editing de WordPress. Pour personnaliser votre site :

1. Allez dans Apparence > Éditeur
2. Personnalisez les modèles, les parties de modèles et les styles globaux
3. Utilisez les compositions prédéfinies pour construire rapidement vos pages

== Prérequis ==

* WordPress 6.6 ou supérieur
* PHP 7.4 ou supérieur
* Un navigateur moderne

== Développement ==

Pour le développement avec Tailwind CSS :

1. Installez Node.js et npm
2. Exécutez `npm install` dans le dossier du thème
3. Utilisez `npm run dev` pour le développement
4. Utilisez `npm run prod` pour la production
